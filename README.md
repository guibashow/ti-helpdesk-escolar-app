# 📱 TI Help Desk Escolar

Protótipo de aplicativo mobile para gerenciamento de chamados de TI em ambiente escolar.

## 🎯 Objetivo
Desenvolver uma solução simples para registrar e organizar atendimentos técnicos dentro de uma instituição de ensino.

## 🚀 Funcionalidades
- Tela de login
- Abertura de chamados
- Listagem de chamados
- Interface mobile responsiva

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript
- Simulação de aplicação mobile

## 📱 Demonstração
O sistema simula um aplicativo Android, podendo ser acessado diretamente pelo navegador.

As imagens do funcionamento do sistema podem ser colocadas na pasta `/prints`.
